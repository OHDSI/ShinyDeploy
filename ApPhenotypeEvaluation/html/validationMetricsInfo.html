Description

A table showing the measurement error metrics from the validation study. For each database, the table shows:

  • TP: True positives
  • TN: True negatives
  • FP: False positives
  • FN: False negatives.
  • Sens: Sensitivity
  • PPV: Positive predictive value
  • Spec: Specificity
  • NPV: Negative predictive value
  • Prev: Prevalence

Options

Select the database and cohort to explore in the side bar.
